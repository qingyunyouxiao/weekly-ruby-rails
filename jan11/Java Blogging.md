该仓库中的代码用于在撰写关于 Java 日志记录和异常处理的博客文章时演示。

使用这些特定示例的博客文章如下：

+ [Java 日志教程：帮助你入门的基本概念](https://sematext.com/blog/java-logging/)
+ [Java 日志最佳实践：你应该知道的 10 个技巧，以充分利用日志](https://sematext.com/blog/java-logging-best-practices/)

## 代码示例
提供以下代码示例：

1. configjul - java.util.logging 配置示例
2. exceptions - 与异常相关的示例
3. log4j2 - 基于 Log4j 2 的示例
4. log4j2json - 使用 JSON 日志的 Log4j 2 示例
5. log4jmasking - 带有屏蔽功能的 Log4j 2 示例
6. log4jslf4j - Log4j 与 SLF4J 示例
7. logback - 基于 Logback 的示例
8. performance - 异常性能比较
9. simple - 使用 java.util.logging 和 SLF4J 进行纯 Java 日志记录，无需额外的日志框架
10. slf4j - 基于 SLF4J 的示例
11. logging_with_log4j - 《使用 Log4j 进行日志记录》博客文章的示例
